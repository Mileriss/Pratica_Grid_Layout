Grid Layout

Grid Layout permite organizar os elementos de uma página HTML dentro de "grades", onde é possível manipular e posicionar as grades da forma que desejar.
O uso do Grid é bem parecido com o Flex e ambos compartilham dos mesmos atributos para organização (align-items, justify content e etc...)

Permite que você determine a Quantidade e Tamanho das colunas

Neste exemplo será utilizada 3 colunas com o tamanho:auto (padrão)
